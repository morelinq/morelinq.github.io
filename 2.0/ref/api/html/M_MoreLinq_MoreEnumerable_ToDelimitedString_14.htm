Usage Note
In Visual Basic and C#, you can call this method as an instance method on any object of type IEnumerable<(Of <(<'Int64>)>)>. When you use instance method syntax to call this method, omit the first parameter.
Remarks
This operator uses immediate execution and effectively buffers the sequence.
