ScanRightTSource(IEnumerableTSource, FuncTSource, TSource, TSource)
Peforms a right-associative scan (inclusive prefix) on a sequence of elements. This operator is the right-associative version of the ScanTSource(IEnumerableTSource, FuncTSource, TSource, TSource) LINQ operator.
ScanRightTSource, TAccumulate(IEnumerableTSource, TAccumulate, FuncTSource, TAccumulate, TAccumulate)
Peforms a right-associative scan (inclusive prefix) on a sequence of elements. The specified seed value is used as the initial accumulator value. This operator is the right-associative version of the ScanTSource, TState(IEnumerableTSource, TState, FuncTState, TSource, TState) LINQ operator.
